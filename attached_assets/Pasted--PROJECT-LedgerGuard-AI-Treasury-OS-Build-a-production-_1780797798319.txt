# PROJECT: LedgerGuard AI Treasury OS

Build a production-grade full-stack SaaS application called LedgerGuard.

LedgerGuard is an AI-powered treasury operating system for Solana organizations, startups, DAOs, communities, and businesses.

The core principle of the product is:

"AI can prepare transactions. Only Ledger can authorize them."

The application must be fully functional on Solana Devnet and capable of moving to Mainnet simply by changing environment variables.

This is NOT a mockup project.

Build a working application with real wallet connections, real transaction creation, real transaction simulation, real transaction broadcasting, database persistence, AI-powered transaction generation, policy validation, audit logging, and Ledger integration readiness.

---

# PRIMARY USER JOURNEY

1. User connects wallet.
2. LedgerGuard loads treasury information.
3. User chats with AI.
4. AI generates treasury actions.
5. Policy engine validates action.
6. Risk engine scores transaction.
7. Transaction is created.
8. User reviews transaction.
9. Transaction is signed through Ledger.
10. Transaction is broadcast to Solana.
11. Audit logs are stored.
12. Dashboard updates automatically.

# ENVIRONMENT CONFIGURATION

Create environment switching.

Example:

NEXT_PUBLIC_SOLANA_NETWORK=devnet

RPC_URL=https://api.devnet.solana.com

OPENAI_API_KEY=

DATABASE_URL=

LEDGER_MODE=speculos

When switched to:

NEXT_PUBLIC_SOLANA_NETWORK=mainnet-beta

The application should automatically operate on mainnet without code modifications.
